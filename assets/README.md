# Assets

Source-of-truth assets live here. Keep this folder organized so designs map 1:1 to shipped UI.

## Structure
- images/ � App icons, splash assets, and existing exported raster images.
- icons/ � Custom SVGs not covered by Phosphor.
- fonts/ � Only for licensed/custom fonts (Plus Jakarta Sans is loaded via Expo Google Fonts).
- illustrations/ � Static SVG/PNG artwork.
- animations/ � Lottie/Rive files and exports.
- brand/ � Logos, app icon sources, and export notes.
- placeholders/ � Temporary assets for layout/testing.

## Placeholders
Use lightweight placeholders when the final asset isn't ready.
Name format: `placeholder-<feature>-<purpose>.<ext>` (example: `placeholder-today-hero.png`).
Document the expected replacement in the folder README.
