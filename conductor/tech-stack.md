# Technology Stack\n\n## Core\n- **Framework:** React Native with Expo (SDK 54) - Provides a robust cross-platform mobile development environment.\n- **Routing:** Expo Router (v6) - File-based routing for React Native apps.\n- **Language:** TypeScript - Ensures type safety and improves developer productivity.\n\n## Styling & UI\n- **Styling Framework:** NativeWind (v4) - Tailwind CSS integration for React Native.\n- **Fonts:** Expo Google Fonts (Inter, Lato, Playfair Display).\n- **Icons:** Expo Vector Icons and Expo Symbols.\n\n## State & Data\n- **Persistence:** @react-native-async-storage/async-storage - For secure, local data storage.\n- **Architecture:** Separation of Concerns (SoC) using custom Hooks for orchestration and UI components for rendering.\n\n## Services\n- **AI Integration:** Centralized in the `services/` layer (e.g., `ai.ts`) for modularity.\n- **Journaling Logic:** `services/journalStorage.ts` for managing entry I/O.\n\n## Testing & Quality\n- **Test Runner:** Jest.\n- **Testing Library:** React Native Testing Library.\n- **Linting:** ESLint with Expo configuration.
