# Specification: AI-Driven Reflection\n\n## Overview\nThis track implements the primary conversational interface where users interact with an AI to log journal entries and receive reflective prompts.\n\n## Requirements\n- **Conversational UI:** A chat-like interface following the `example-design` styles.\n- **AI Persona:** Supportive, empathetic therapist tone that avoids sycophancy.\n- **Integration:** Connect the frontend UI to the existing `services/ai.ts`.\n- **Storage:** Persist chat history and generated journal summaries to local storage.\n- **TDD:** All new logic must be covered by Jest tests before implementation.
